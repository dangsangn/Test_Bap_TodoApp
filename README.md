# Introduction
this is User-Interface of Todo App, using ReactJS Typescript Vite

## How to use

Clone the repository.

**Run Server:
```sh
yarn server
```
**Run Client:

Install:

```sh
yarn install
```

Start dev: 

```sh
yarn dev
```

Build:
```sh
yarn build
```
Note:
When update status task, please double click task you want to update.
